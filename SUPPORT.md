# Support

Coding Autopilot System is a public engineering portfolio maintained on a best-effort
basis.

## Where To Ask

- Use a repository issue for reproducible defects and scoped feature requests.
- Use repository discussions, when enabled, for design questions and general help.
- Follow the repository README and runbooks before requesting support.
- Report vulnerabilities privately according to [SECURITY.md](SECURITY.md).

## Include With Requests

- repository, branch, commit, and operating system;
- exact command or workflow executed;
- expected and actual behavior;
- sanitized logs and minimal reproduction steps;
- validation already attempted.

Remove credentials, tokens, personal data, proprietary code, and sensitive prompts.

## Support Boundaries

Maintainers cannot provide guaranteed response times, private consulting, production
incident response, or support for undocumented modifications. Requests may be closed
when they are unsafe, incomplete, unreproducible, or outside the repository's scope.
